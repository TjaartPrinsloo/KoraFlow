{{ _("3. Schedule 4 Classification") }}

{{ _("Schedule 4 medications may only be prescribed by a medical practitioner, dentist, nurse, or other authorized prescriber registered with the Health Professions Council of South Africa (HPCSA) or South African Nursing Council (SANC).") }}

{{ _("As Schedule 4 substances, GLP-1 medications:") }}

  • {{ _("Require a valid prescription for each dispensing") }}
  • {{ _("Cannot be sold over-the-counter") }}
  • {{ _("Must be dispensed by a registered pharmacist") }}
  • {{ _("Are subject to record-keeping requirements") }}

{{ _("5. Important Safety Information") }}

{{ _("Contraindications") }}

{{ _("GLP-1 medications should NOT be used if you have:") }}

  • {{ _("Personal or family history of medullary thyroid carcinoma (MTC)") }}
  • {{ _("Multiple Endocrine Neoplasia syndrome type 2 (MEN 2)") }}
  • {{ _("Known hypersensitivity to the active substance") }}
  • {{ _("Pregnancy or breastfeeding") }}

{{ _("Common side effects may include:") }}

  • {{ _("Nausea and vomiting") }}
